# Resume Project Summary

## Project Title

Synthetic US Tax Dataset Generator

## One-Line Description

Built a Python-based document automation system to generate synthetic US tax datasets with supporting documents, executive summaries, and official-template PDF form filling.

## Resume Bullets

- Developed a synthetic tax dataset generation pipeline covering 2,000 tax cases across California, Texas, New York, Illinois, and Florida for tax years 2020 through 2025.
- Automated the creation of questionnaires, supporting tax documents, executive summaries, and federal/state tax form outputs with cross-document consistency checks.
- Implemented PDF field-filling workflows for federal, state, and supporting tax templates using Python, pypdf, and template field mapping.
- Built batch-generation and packaging utilities to support difficulty-tiered scenarios, manifest generation, and weekly delivery workflows.
- Added a Streamlit app layer to demo small-batch generation interactively from the GitHub-hosted project.
