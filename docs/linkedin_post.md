Built and published a Synthetic US Tax Dataset Generator project focused on large-scale document automation.

Highlights:
- generated synthetic individual tax scenarios across California, Texas, New York, Illinois, and Florida
- covered tax years 2020 through 2025
- supported Level 1, Level 2, and Level 3 case complexity
- produced questionnaire packets, supporting documents, executive summaries, and tax-form outputs
- added official-template field filling for the blank federal, state, and supporting tax forms collected for the project
- wrapped the project in a Streamlit app for interactive small-batch demos
- published the codebase on GitHub for portfolio and review use

This project pushed me to work across data modeling, document consistency, PDF automation, batch generation, and delivery packaging at scale.

A big takeaway from this build was how important validation and cross-document consistency are when generating structured synthetic datasets. Getting values to stay aligned across W-2s, 1099s, Form 1040 workflows, schedules, state returns, and summaries was the real challenge.

Grateful for the learning that came from building an end-to-end workflow instead of only generating mock records.
